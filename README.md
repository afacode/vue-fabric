# vueFabric
拼图基于fabric.js

### 技术栈
> `vue全家桶+axios+sass+canvas(fabric.js)+element-ui+七牛云等`

### 运行
```js
dev
npm run dev or npm run start

build prod
npm run build:prod

build test
npm run build:test
```

### 实现的功能
>  图片的复制粘贴，删除，隐藏，翻转，历史记录，回退撤销，锁定，图片组合打散，剪切，图层管理，图片滤镜，图片导出，案例保存，加载，收藏夹，个人中心功能等
验证码，微信登陆
